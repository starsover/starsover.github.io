【置顶】别再去下什么2345、360了，这些才是装机必备！

2022-03-26

最近看到别人装机给别人装了一大堆乐色软件。。。搞得我血压都高了。。。

一、系统镜像

  • MSDN，我告诉你（旧站/新站）

    MSDN，我告诉你中收集了许多系统，且都是原版，无捆绑。且已稳定运行多年。

    （注：①MSDN，我告诉你中的系统下载方式为BT或ED2K下载，需要用到下文中的迅雷。②最近在搜索引擎上出现了很多MSDN，我告诉你的冒牌站，请注意辨别。）

  • Microsoft官方下载（Windows 11/Windows 10/Windows 8.1/Windows 7）

    （十分简单，不用描述。）

二、PE系统

  • 百科

    Windows PE是简化版的Windows 或 Windows Server，放在一个可直接启动的U盘或者CD/DVD光盘中，特点是启动时出现 Windows 的图形界面（GUI）

  • ①FirPE（发布页面）

    优点：能联网、功能多、Windows 11内核、能装在磁盘中、自带的Easy RC很好用。

    缺点：体积稍大

  • ②微PE工具箱（WePE）（官网）

    优点：功能多且实用、能装在磁盘中、体积较小。

    缺点：不能联网。

  • ③优启通（EasyU）（官网）

    优点：能联网、自研功能多、能装在磁盘中。

三、压缩软件

  • 7-Zip（English/中文）

    7-Zip 是一款拥有极高压缩比的开源压缩软件。

    功能：

    • 使用了 LZMA 与 LZMA2 算法的 7z 格式 拥有极高的压缩比
    • 支持格式：
      • 压缩 / 解压缩：7z、XZ、BZIP2、GZIP、TAR、ZIP 以及 WIM
      • 仅解压缩：AR、ARJ、CAB、CHM、CPIO、CramFS、DMG、EXT、FAT、GPT、HFS、IHEX、ISO、LZH、LZMA、MBR、MSI、NSIS、NTFS、QCOW2、RAR、RPM、SquashFS、UDF、UEFI、VDI、VHD、VMDK、WIM、XAR 以及 Z
    • 与 ZIP 及 GZIP 格式相比，7-Zip 能提供比使用 PKZip 及 WinZip 高 2-10% 的压缩比
    • 为 7z 与 ZIP 提供更完善的 AES-256 加密算法
    • 7z 格式支持创建自释放压缩包
    • Windows 资源管理器集成
    • 强大的文件管理器
    • 强大的命令行版本
    • 支持 FAR Manager 插件
    • 支持 87 种语言

四、下载软件

  • ①Neat Download Manager（NDM）（English）

    NDM与IDM的原理一样，都是利用多线程下载以提高下载速度。

    但与IDM不同的是：NDM它是免！费！的！！！

    缺点：NDM官方只有英文版，但各大下载站中有中文补丁。但图标足够国人辨识和使用了。

  • ②迅雷

    不建议去下载迅雷官网上的，因为那玩意儿广告多占用大。。。

    可以去看看另一位博主：阿虚同学的一篇文章：

    测试了15款迅雷，最好用的竟然是这个！

五、杀毒软件

  • ①Windows Defender（Windows自带）

    其实Windows自带的杀毒软件已经够用了，现在已经没有必要再另外安装了。

  • ②火绒（中文）

    当然，如果你实在不放心的话，可以去装一个火绒，占用不太大，还好用！

真的不要去下360、2345和什么XX电脑管家了！！！

六、浏览器

  • ①奔跑中的奶酪（running cheese）定制版浏览器

    另一位博主奔跑中的奶酪推出的Edge、Chrome、Firefox的绿色便携版，拥有更多功能，小白推荐下载这个。

  • ②Microsoft Edge（Windows自带）

    这是微软新推出的一款浏览器，与前身IE不同，它使用的是chromium内核。没错，与谷歌的Chrome一样。

    也就是说，你可以安装Chrome的拓展。

    如果你是Windows 10及以上，那么你不需要下载。它就捆绑在你的电脑中。

    但如果你是Windows 10一下或没有Microsoft Edge，你也可以自己去Microsoft Edge官网上下载。

  • ③Google Chrome（中文/你无法访问的English）

    咕噜咕噜推出的一款简洁而先进的浏览器，使用自家的chromium内核。能浏览大部分网页。

  • ④firefox（火狐浏览器）（国内版/国际版）

    Mozilla推出的一款简洁而先进的浏览器，使用Gecko内核。能浏览大部分网页。

七、修图软件

  • Adobe Photoshop

    如果你没钱的话，可以去下[vposy](https://m.weibo.cn/u/1112829033)大佬的破解版（注：需要关注微博后才能看到下载链接。）

    如果你既没钱又没性能的话，你可以使用来自另一个博主阿虚同学的CC 一键安装版

什么？？？Windows密码又忘了？！还好，这次不用去电脑店！

· 2022-05-02 · # 电脑 # Windows # 软件 # 推荐

你有没有这样的经历？

你在收拾家里的杂物时，意外翻到了以前的老电脑，就当你满怀当年的记忆打开电脑时，却发现：当年我竟TM设了密码？！然后当你努力回想密码是什么时，却发现——当年的记忆压根就没有什么密码！！！

（注：本篇文章不是讲被BitLocker锁住的盘密码忘了而导致无法访问或者开不了机的问题。如果你BitLocker密码忘了且没有恢复密钥的话，你就只能和那块盘说拜拜了。连电脑店师傅也救不了你的盘！）

一、NTPWEdit（民间的Windows密码修改）（官网）

你可能已经在各个PE系统中见过它了

它是一位由俄国大佬“Vadim Druzhin”制作

支持Windows 2000，XP，Vista，7，8，10系统

不支持解密密码或更改域、Active Directory 或 Microsoft 帐户密码。

NTPWEdit通过直接修改文件 C:\WINDOWS\SYSTEM32\CONFIG\SAM 来更改密码。运行时，Windows会阻止对该文件的任何访问，因此必须在PE上执行它。

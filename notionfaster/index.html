2、Android

（在进行下文操作之前，您必须保证自己拥有root权限。）

打开：

/system/etc/hosts

那么，接下来的步骤就和Windows的一样了！

二、修改DNS

将你的 DNS 修改为以下 IP 即可：

119.29.29.29
114.114.114.114

三、Cloufare IP 段

108.162.236.1/24 联通 走美国
172.64.32.1/24 移动 走香港
104.16.160.1/24 电信 走美国洛杉矶
172.64.0.0/24 电信 美国旧金山
104.20.157.0/24 联通 走日本
104.28.14.0/24 移动 走新加坡

104.16.160.1/24 比如这个 IP 段，是包含了104.16.160.1～～104.16.160.255 之间的所有 IP，你得 ping 这所有 IP 看他们的连接怎样，然后筛选出最好的添加到Hosts 里，windows 推荐一个免费的绿色软件 --- pinginfoview 可以直接一口气ping 这所有 IP。--------- Literal Tomb
